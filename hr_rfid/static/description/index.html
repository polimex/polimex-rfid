Fully Functional Professional RFID Access Control System

Enhance your Odoo experience with our professional RFID system. Gain information about the entrance of the company’s building/rooms or specific places

Features

General

  • Gain information about the entrance of the company’s building/rooms or specific places
  • Multi site and multi company support
  • Fully integrated in Odoo models
  • Easy to install and support
  • Full remote support of the installed equipment on the customer site
  • Cloud based installation for Installing companies
  • Part of a existing Odoo installations
  • Local installation on VM (polimex.cloud)
  • Demo installation (demo.polimex.co)
  • Code available in Github (github.com/polimex)

Equipment

  • Multi technology readers support Wiegand 26/32/34 bit
  • Reader modes: Card Only, Card and PIN, Card or Work code
  • Local and Global Anti Pass Back
  • Duress mode
  • Setting Time schedules
  • Global and Local emergency/Fire distribution
  • Dual person mode
  • Interlock management
  • Turnstile ready
  • Vending Machine support with Executive protocol
  • Fully open equipment SDK/API
  • Full support of all Polimex manufactured controllers

Module Features

  • Fully integrated in core Employee module
  • Unlimited Access groups with inheritance
  • Real-time user events
  • Temperature and Humidity history
  • Cards for Employee and for Partners
  • Control the access of each department in the company
  • Department manager role
  • Give expiration date of a given access group
  • Import cards and their owners using excel, or csv files, or create them manually
  • Emergency Zone management
  • Attendance Zone management on door, department, etc.

Sub-module Addons

  • Base PMS module vertical
  • Attendance calculation
  • Hourly cost & work time
  • Portal integration
  • Barcode Generation and partner share
  • Vending product sales with Inventory option
  • RFID Voting module vertical
  • RFID Services sales and management
